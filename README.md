## Getting Started
### Prerequisites
* Windows or Mac OS
* [Adobe After Effects CC 2017+](https://www.adobe.com/)
### Clone this repo
```bash
git clone https://github.com/HypoX64/Plogo
cd Plogo
```
### Open main.aep
```bash
main.aep
```
